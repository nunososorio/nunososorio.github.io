Pandas Cheat Sheet for Beginners

Pandas Functions

1. pd.read_csv()

Reads a CSV file into a DataFrame.

import pandas as pd
df = pd.read_csv('file.csv')
print(df.head())

2. pd.concat()

Concatenates pandas objects along a particular axis.

df1 = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
df2 = pd.DataFrame({'A': [5, 6], 'B': [7, 8]})
df = pd.concat([df1, df2], axis=0)
print(df)

3. pd.merge()

Merges DataFrames based on keys or indexes.

df1 = pd.DataFrame({'key': ['A', 'B', 'C'], 'value1': [1, 2, 3]})
df2 = pd.DataFrame({'key': ['A', 'B', 'D'], 'value2': [4, 5, 6]})
df = pd.merge(df1, df2, on='key')
print(df)

Pandas Methods

1. df.head()

Displays the first n rows of the DataFrame.

print(df.head(10))

2. df.drop()

Drops specified labels from rows or columns.

df = df.drop('column_name', axis=1)
print(df)

3. df.groupby()

Groups the DataFrame using a mapper or by a Series of columns.

grouped_df = df.groupby('column_name').mean()
print(grouped_df)

4. df.describe()

Generates descriptive statistics for numeric columns.

print(df.describe())

5. df.info()

Provides a concise summary of the DataFrame, including the data types and non-null values.

print(df.info())

6. df.fillna()

Fills NA/NaN values using the specified method.

df = df.fillna(0)
print(df)

7. df.dropna()

Removes missing values.

df = df.dropna()
print(df)

8. df.astype()

Casts a Series to a specified dtype.

df['column_name'] = df['column_name'].astype(float)
print(df)

9. df.sort_values()

Sorts a DataFrame by the values along either axis.

df = df.sort_values(by='column_name')
print(df)

10. df.rename()

Renames the labels in the DataFrame.

df = df.rename(columns={'old_name': 'new_name'})
print(df)
